    二叉树图例

    1.前序遍历

    前序遍历为从父节点开始读取,每次优先读取左边的子节点依次输出

    那么以上图做为例子

    前序遍历则是 A -> B -> D -> E -> C -> F

    2.中序遍历

    中序遍历是二叉树中比较重要的一种遍历方法

    首先要找到二叉树最左侧的节点,我们可以从A节点开始看

    A节点有左侧节点B, 那么需要先输出B

    然后看B节点,B节点也有左侧的节点,因此同样先输出B的左侧节点D

    左侧节点输出完后接上根节点,再者就开始依次输出右侧节点,右侧节点的输出依旧是左侧的子节点优先输出

    最后的结果是 D -> B -> E -> A -> F -> C (PS: F是C的左节点,因此在C的前面输出)

    3.后序遍历

    后序遍历遵循 左子树,右子树,根节点

    图例的结果则是: D -> E -> B -> F -> C -> A

    遍历每个节点时,都优先输出这个节点的左子树和右子树,最后才是这个节点
